# Portfolio Website – SRS (Software Requirements Specification)

## 1. Introduction

### 1.1 Purpose

This document defines the requirements for a personal portfolio website that showcases the user’s profile, skills, projects, experience, and certifications.

### 1.2 Scope

The website will be a responsive, visually appealing, and interactive platform with animations, modern colors, and typography.

### 1.3 Objectives

* Create a professional online presence
* Showcase skills and projects
* Improve job opportunities

---

## 2. Overall Description

### 2.1 Product Perspective

A single-page or multi-page web application accessible on desktop and mobile devices.

### 2.2 User Classes

* Recruiters
* Hiring managers
* General visitors

### 2.3 Operating Environment

* Web browsers (Chrome, Edge, Safari)
* Devices: Mobile, Tablet, Desktop

---

## 3. Functional Requirements

### 3.1 Home Page

* Display name and tagline
* Hero section with animation
* Navigation bar

### 3.2 About Section

* Personal introduction
* Education details
* Profile image

### 3.3 Skills Section

* List of technical and soft skills
* Progress bars or animated charts

### 3.4 Projects Section

* Project cards with:

  * Title
  * Description
  * Technologies used
  * Live/demo links

### 3.5 Experience Section

* Work or internship experience
* Timeline layout

### 3.6 Certificates Section

* Display certificates
* Download/view option

### 3.7 Contact Section

* Contact form
* Email and phone details

---

## 4. Non-Functional Requirements

### 4.1 Performance

* Fast loading time (<3 seconds)

### 4.2 Usability

* Easy navigation
* Clean UI/UX

### 4.3 Responsiveness

* Fully responsive design

### 4.4 Security

* Form validation
* No sensitive data exposure

---

## 5. Design Requirements

### 5.1 Colors

* Primary: #6C63FF (Purple)
* Secondary: #F5F5F5 (Light background)
* Accent: #FF6584 (Pink highlight)

### 5.2 Fonts

* Heading: Poppins
* Body: Open Sans

### 5.3 Animations

* Smooth scrolling
* Fade-in effects
* Hover animations on buttons/cards
